Generated Excel Document Architecture

The downloadable Excel file is organized into 4 sheets, designed to mimic a professional enterprise BI report:

Sheet 1

Raw Data

Contains the original raw CSV data as uploaded. The header row is styled with a professional dark blue background (#1F4E78) and white bold font. Gridlines are explicitly enabled, columns are auto-fit to cell sizes, and basic columns are aligned appropriately.

Sheet 2

Cleaned Data

Contains the polished sales dataset after running the automated cleaning modules. Records with nulls in crucial ID or value columns are filtered out, duplicate orders are dropped, dates are standardized to YYYY-MM-DD, and text casings are corrected.

Sheet 3

Summary Report

A beautifully formatted dashboard sheet containing: a styled side-by-side KPI Summary block, a Top 10 Products by Revenue table, a Monthly Revenue Trend table, and a Region-wise Sales Breakdown table. Includes automatic summary rows using native Excel formulas (e.g. SUM(...)).

Sheet 4

Charts

Houses high-resolution, presentation-grade vector charts generated during the pipeline run using Matplotlib. These include a Top 5 Products Bar Chart, a Monthly Revenue Trend Line Chart, and a Regional Share Donut Chart, embedded directly at specific anchors.
